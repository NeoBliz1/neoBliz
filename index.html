After studying the general features of front-end development, I decided to try a new direction of programming - data science engineering. After I studied the Machine Learning course at Codecademy. It was positive experience, but I preferred continue web development way.

freeCodeCamp

Searched the Internet for a while, I found freeCodeCamp.org. This is a useful training center that gives knowledge for free. I was crazy about interesting interactive lessons. I studied JavaScript in depth for about 300 hours. At the end of the training block, I received a Certificate.

  • Bootstrap framework;
  • jQuery library;
  • SASS preprocessor;
  • React, a popular JS framework from Meta.
  • Redux state management library
  • A block of React and Redux that looks at their interaction.

At the end of the course it is proposed to create five small applications based on the knowledge gained. I've complete it successful. I've got Certificate.
